Please forgive our tardiness! The API docs will be available soon! In the meantime, you can create them yourself. How?

  1. Make sure you have installed Apache Maven, see here.
  2. Download the DataExpress source code from our github project.
  3. Assuming you downloaded the source code to your home directory as DataExpress, from a command prompt, execute the following commands: 
$cd ~/DataExpress
$mvn site
If successful, there should now be an index.html file in the directory ~/DataExpress/target/site. Simply double click this file to open it in your default web browser. There will be a link on the left called Project Reports. Click that link to and then click the link for ScalaDocs.
